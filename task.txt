1. შექმენი user ობიექტი რომელზეც შესაძლებელი იქნება API call-ის გამოყენება (შეგიძლია გამოიყენო json-server https://www.npmjs.com/package/json-server).
2. API data-ს დახმარებით შექმენი table, დიზაინის მიხედვით.
3. უნდა შეიძლებოდეს მომხმარებლის დამატება, წაშლა და რედაქტირება API call-ების საშუალებით.
4. უნდა შეიძლებოდეს მომხმარებლის გაფილტვრა (search ველით) და დასორტვა(user, role ან status-ის მიხედვით).
5. მომხმარებლის რედაქტირებისას უნდა გამოიყენო როუტინგი (React router + react portal).
6. ყველა ფორმას უნდა ქონდეს ვალიდაცია (შეგიძლია გამოიყენო დამხმარე library. მაგალითად: formik, react-hook-form ან ნებისმიერი, სურვილისამებრ)
7. UI-ს შესაქმნელად შეგიძლია გამოიყენო ნებისმიერი ui library (მაგალითად: Tailwind, bootstrap). შეგიძლია xd-ში ნახო პროტოტიპი, რომელიც გაგიიოლებს ინტერფეისის შექმნას.
